
# 🌆 Metro City Gaming Website

A modern, synthwave-inspired site for the **Metro Gaming Community**, supporting:

🎮 FiveM | 🧠 CS2 | 💬 Discord

---

## 🚀 Features

- 🎨 Modern landing page with dark theme
- 📡 Live server info for FiveM & CS2
- 🔗 Discord join + verification
- 📃 RP rules & community guidelines
- 🧑‍💼 Staff directory
- 📢 News & Events
- 🧠 Whitelist job applications
- 🛒 Virtual RP Shop
- 📅 Event calendar & RSVP
- 💬 Live chat via Discord or Tawk.to
- 🎛️ Theme selector (Cyber, GTA, Classic)

---

## 📦 Deployment

This site is static and deployable on **[Vercel](https://vercel.com)** in 3 steps:

1. Upload all contents of this folder to GitHub
2. Connect GitHub to Vercel
3. Deploy — you're live in 1 minute!

---

## 🧠 Tech Stack

- HTML5, CSS3 (Tailwind-ready)
- JavaScript (LocalStorage, basic logic)
- Mobile responsive
- Ready for Discord embeds & Steam auth
